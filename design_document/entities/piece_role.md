# PieceRole Enum

PieceRole is to define the type/role of a chess piece.

## Enumerators

Each name describes the role name, and each value describes the value of the role in board. The value can be used in suggestion systems, etc.. 

*   KING   = 10000
*   QUEEN  = 9
*   BISHOP = 7
*   ROOK   = 5
*   KNIGHT = 3
*   PAWN   = 1
